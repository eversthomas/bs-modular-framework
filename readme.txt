Tabellenbasiertes, modulares Datenframework für strukturierte Inhalte im WordPress-Backend.

== Description ==

BS_Modular-Framework ist ein tabellenbasiertes Datenframework für strukturierte Inhalte im WordPress-Backend.
Es ermöglicht Administratoren, Module und deren Felder zu definieren und Redakteuren, Einträge zu diesen
Modulen zu pflegen. Dieses MVP konzentriert sich ausschließlich auf die Admin-Verwaltung ohne Frontend-Ausgabe.

== Installation ==

1. Plugin-Verzeichnis `bs-modular-framework` in das Verzeichnis `/wp-content/plugins/` hochladen.
2. Plugin über das WordPress-Backend unter „Plugins“ aktivieren.
3. Das neue Admin-Menü „Modular Framework“ im Backend aufrufen.
